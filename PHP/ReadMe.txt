1. Make sure you have PHPMailer installed.
      For Windows User: 
                    Download the folder and extract all its files to your location.
                    location : https://github.com/PHPMailer/PHPMailer
      For Linux Users:
                    pip install composer.
                    composer require phpmailer/phpmailer
                    
2. Since Gmail Do not allow local devices to access so change your settings : 
            Manage Account --> Security --> Turn on 2 step Verification --> Generate App Password
            
3. You can now use this generated password to send emails.
